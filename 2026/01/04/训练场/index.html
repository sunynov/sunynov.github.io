如何通过异常对象拿到执行命令的能力

在 Python 中，__traceback__ 是一个特殊的属性，它存在于 异常对象（Exception objects） 中。简单来说，它是异常发生时的“现场快照”，记录了程序崩溃时的调用栈信息。

当 Python 抛出一个异常时，它会为该异常关联一个 traceback 对象。这个对象存储在异常实例的 __traceback__ 属性里。

traceback 对象本身是一条 链表。你可以通过它不断向上或向下寻找函数帧（Frame），从而获取本不该获取的变量。

它包含以下关键子属性：

  • tb_next: 指向调用堆栈中的下一个（更深层的）追踪对象。
  • tb_frame: 指向当前追踪对象对应的 栈帧对象（Frame object）。
  • tb_frame.f_locals: 这是 CTF 的核心！ 它是一个字典，包含了该栈帧中所有的局部变量。
  • tb_frame.f_builtins 是栈帧对象（Frame object）中的一个属性，它是一个字典，包含了当前栈帧环境下可以使用的所有内建函数和异常。

也就是说我们可以用下面这条链子拿到import

如何绕过字符数量限制

在 Python 中，str.format() 是一种强大的字符串格式化方法。str.format() 有一个非常有杀伤力的特性：它可以访问对象的属性（Attribute Access）。在 SSTI 场景下，如果你能控制格式化字符串（花括号里的内容），即使你不能直接执行代码，你也可以通过 format 泄露内存中的全局变量和配置信息。

当你写： "{0.__traceback__.tb_frame.f_builtins[eval]}" 你实际上是获取了 eval 这个函数对象。

解决方法

make_set(bits, str1, str2…)

make_set函数返回一个字符串，根据bits来选取后面的字符串参数，如果某一位是1，就选中对应的字符串

比如SELECT make_set(5, 'a', 'b', 'c', 'd');

5转换成二进制是0101，从右向左是1,0,1,0，因此a和c会被选中，然后用逗号连接并输出'a,c'

quote(str)

quote会返回被单引号包裹的字符串并且能自动转义里面的特殊字符

quote(‘hello’) 输出'hello'

quote(“it’s test”) 输出'it\'s a test'
